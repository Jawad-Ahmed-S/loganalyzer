# Log Analyzer

##clone 
```bash
git clone https://github.com/Jawad-Ahmed-S/loganalyzer
```

## Setup

```bash
pip install -r requirements.txt
```

## Run

```bash
python run.py path/to/logfile.log
```

## Generate Sample Log

```bash
python scripts/generate_log.py
python run.py scripts/sampleLog.txt
```
